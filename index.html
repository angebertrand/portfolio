Ce jeu est donc un fangame d'horreur-humoristique pour la chaîne "Wankil Studio - Laink et Terracid". Il est fait via Unreal Engine 5.4.3 en utilisant seulement des Blueprints. Je fus tout seul, j'ai fais les bruits de pas, une partie de la musique, les deux animations de "jumpscare" ci-contre via Blender (avec Blender toujours, j'ai aussi fait de la modélisation 3D afin de modifier les modèles 3D originaux pour les transformer en "monstre")
Il est téléchargable via Gamejolt

Jeu d'infiltration die-and-retry fait sur Unreal Engine 5.2.1 en Blueprint trouvable ici sur Itch.io.
Créé pour le GameJam : Games for Blind Gamers 3, se concentrant sur l'accessibilité des personnes mal-voyantes voire non-voyantes, le principe est simple : vous êtes un robot qui doit obéir et récupérer des documents sans se faire repérer par d'autres rebots ayant des patterns simples. MAIS, vous êtes le premier robot à être conçut sans caméra pour éviter toute distraction.

J'ai fini 19ème sur 26.
